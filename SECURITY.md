# Security Policy

## Supported Versions

This project is actively maintained on the `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report details privately to the maintainers with:

- Vulnerability description
- Steps to reproduce / proof of concept
- Affected files or routes
- Suggested mitigation (if available)

We will acknowledge receipt within 72 hours and provide an update after triage.
